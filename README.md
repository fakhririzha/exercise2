 ## Basic Usage
 ```sh
1 + 1 = 2
4 - 2 = 2
3 * 3 = 9
8 / 4 = 2
10 % 2 = 1
 ```

 ## Keyboard Shorcut
 - `Enter`     => Display Result
 - `0-9`       => Number Input
 - `Delete`    => Delete All input
 - `Backspace` => Delete 1 Number
